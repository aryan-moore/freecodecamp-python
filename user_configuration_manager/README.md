# User Configuration Manager

A Python program to manage user settings, including adding, updating, deleting, and viewing settings.

## Features
- Add a new setting
- Update an existing setting
- Delete a setting
- View all current settings

## Concepts Used
- Python dictionaries
- Functions
- Conditional statements
- Loops
- String manipulation

## How to Run
1. Make sure Python is installed
2. Navigate to the `user_configuration_manager` folder
3. Run:
```bash
python user_configuration_manager.py
